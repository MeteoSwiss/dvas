dvas_recipes.initialize.default_arena_path(recipe)¶

pathlib.Path – the default relative Path.

dvas_recipes.initialize.init_arena(recipe, arena_path=None)¶

Initializes a new dvas prcoessing arena, based on a specific processing recipe.

Parameters

  • recipe (str) – the processing recipe. Must be one of [‘demo’, ‘uaii22’].

  • arena_path (pathlib.Path, optional) – relative Path to the processing arena to initialize. Defaults to ./dvas_recipe_arena.
