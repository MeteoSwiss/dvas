get_document(key)¶

Interpret the generator syntax if necessary and return the config document.

The generator syntax is apply only in OneDimArrayConfig with not empty NODE_GEN. Generator syntax is based on regexpr. Fields which are not generator can contain expression to be evaluated. The syntax used is the one described in ConfigExprInterpreter.eval
